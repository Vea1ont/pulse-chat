# Prometheus + Grafana — шпаргалка

## Что делает Prometheus

Prometheus — система сбора метрик. Он сам ходит к твоим сервисам каждые 15 секунд и спрашивает "сколько запросов пришло, как быстро отвечал". Это называется **pull-модель** — он тянет данные сам, а не сервисы ему отправляют.

---

## Как сервис отдаёт метрики

В каждом FastAPI-сервисе подключается библиотека `prometheus-fastapi-instrumentator`. Она добавляет маршрут `/metrics` и встраивается как **middleware** (прослойка).

**Важное заблуждение:** instrumentator не вызывает твои ручки сам. Он просто перехватывает каждый входящий запрос — до и после твоего кода — и фиксирует факт:

```
запрос пришёл
     ↓
instrumentator: засёк время старта
     ↓
твой код отработал
     ↓
instrumentator: +1 к счётчику, записал время ответа
     ↓
ответ ушёл клиенту
```

---

## Где хранятся данные

**В RAM сервиса** — instrumentator держит счётчики прямо в памяти процесса. `/metrics` — это не база данных, это просто ручка которая читает эти переменные из памяти и форматирует в текст. Перезапустил сервис — счётчики обнулились.

**В Prometheus** — настоящая база данных с временны́ми рядами. Он забирает текст с `/metrics` каждые 15 секунд и сохраняет с меткой времени на диск (`prometheus_data`). Хранит 7 дней (настроено в docker-compose.yml).

```
RAM сервиса → /metrics (текст) → Prometheus (диск, 7 дней)
```

---

## Что настраивается где

**`infra/prometheus.yml`** — всё что касается самого Prometheus: куда ходить, как часто, какие метки добавлять. Сервисы там просто перечисляются как цели.

**Код сервиса** — только подключить instrumentator. Больше ничего. Prometheus сам найдёт `/metrics`.

---

## Grafana

Grafana не хранит данные — она хранит только **конфигурацию дашбордов** (что рисовать и как). Данные всегда берёт из Prometheus по запросу.

```
Prometheus (данные) ← Grafana (конфиг дашбордов)
                              ↓
                        графики в браузере
```

Удалишь `prometheus_data` — данных нет, дашборды пустые.
Удалишь `grafana_data` — данные есть, но дашборды придётся создавать заново.

---

## Полная цепочка

```
твой код → instrumentator считает → /metrics в RAM → Prometheus забирает на диск → Grafana рисует
```
